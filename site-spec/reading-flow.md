# Переходы и последовательное чтение

## Поведение при переходе по внутренним ссылкам

При переходе по любой внутренней ссылке:

* должна открываться соответствующая страница справа;
* контент должен прокручиваться к нужному заголовку;
* выбранный пункт в левой навигации должен подсвечиваться;
* родительские уровни в левой навигации должны автоматически раскрываться;
* левая навигация должна прокручиваться к выбранному пункту, чтобы пользователь не терялся после перехода;
* это же поведение должно работать при использовании кнопок браузера назад/вперёд.

При переходе к якорю внутри длинной страницы полезно кратко подсветить или визуально выделить целевой заголовок, чтобы читатель видел место перехода.

## Scrollspy: навигация должна следовать за чтением

При ручном скролле контента активный пункт в левой навигации должен меняться по текущему заголовку.

Левая навигация должна:

* подсвечивать пункт ближайшего текущего заголовка;
* автоматически раскрывать родительские уровни;
* прокручиваться так, чтобы активный пункт оставался видимым.

При обычном скролле контента URL не нужно постоянно переписывать. Иначе история браузера будет засоряться. URL должен меняться при явном переходе по ссылке, но не при каждом изменении активного заголовка из-за чтения.

## Последовательное чтение: кнопка следующей главы

Внизу каждой страницы, кроме последней, должен быть блок перехода к следующей главе.

В блоке должно быть написано:

```text
Следующая глава
<название следующей главы>
```

Название следующей главы должно соответствовать следующему пункту в общей последовательности сайта.

Последовательность чтения:

```text
Orientation
Story_introduction.md
01 Boris Tane
02 Peter Steinberger
03 Simon Willison
04 Arvid Kahl
05 Jökull Sólberg
06 Jesse Vincent
07 HumanLayer
08 Mike McQuaid
09 Calvin French-Owen
10 Mark Erikson
11 Mae Capozzi
12 Matt Pocock
Theoretical_synthesis.md
Cross_story_synthesis.md
Handbook.md
Fieldbook.md
```

Кнопка следующей главы должна вести на верх соответствующей следующей страницы и работать как обычная внутренняя ссылка, чтобы подсветка навигации и браузерные кнопки назад/вперёд работали штатно.
