# Skills

Эта папка содержит повторяемые рабочие режимы. Один skill — один Markdown-файл. Отдельные папки под skill не создаются, пока skill не потребует собственных шаблонов, скриптов или больших вспомогательных материалов.

Skill не заменяет `AGENTS.md`, протоколы и `work/discourse.md`. Он описывает повторяемый режим работы: когда применять, что читать, что делать, где остановиться и что выдать в конце.

Текущие skills:

- `maintain-work-discourse.md` — ведение `work/discourse.md` как однородного дискурса с привязкой к файлам.
- `codex-task-work.md` — базовый режим нетривиальной работы Codex с задачей.
- `chat-github-repo-work.md` — работа ChatGPT с репозиторием через GitHub-интеграцию.
- `finalize-working-branch-for-main.md` — подготовка рабочей ветки к переносу в `main`.
- `stage-0-5-coverage-audit.md` — SDLC artifact coverage и agentic frameworks coverage audit перед writing.
- `restore-anchor-case-from-baseline.md` — восстановление SPDD/Gas Town и других deep anchors из old-site baseline.
- `rollback-working-change.md` — безопасный откат отдельной правки, файла или серии изменений в указанной ветке.
