# Языковые и стилевые правила

Этот файл является коротким входом в распиленный `LangStyleRules.txt`. При правке пользовательского текста нужно читать не только этот файл, но и связанные правила ниже.

Обязательные файлы:

- `russian-language.md` — русский язык, допустимые исключения для английского и запрет английских абзацев.
- `terminology-and-translation.md` — словарь обычных английских слов, которые нужно переводить в русском техническом тексте.
- `human-technical-style.md` — спокойный человеческий технический стиль вместо модельного summary; включает правило о смысловых свёртках, псевдотерминах и механическом разворачивании.
- `english-source-handling.md` — правила переноса материала из англоязычных источников в русский текст.

Если задача касается переноса фактуры из источников или старых черновиков, дополнительно читать:

- `source-and-provenance.md`;
- `content-preservation.md`.
