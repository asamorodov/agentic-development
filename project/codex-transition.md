# Переход к Codex

Codex используется как агент для файловой и проверочной работы: поддержка сайта, перенос материалов по протоколам, создание pass-артефактов, подготовка отчётов и проверка diff.

Разделение ролей:

- ChatGPT / сильная модель: концептуальные решения, критика структуры, проектирование задач и проверка смысловых развилок.
- Codex: работа в репозитории, правки файлов, поддержка сайта, локальные проверки, подготовка отчётов.
- Человек: решения о структуре, merge крупных изменений, принятие содержательных переработок.

Переход выполняется осторожно: сначала осмотр репозитория без правок, затем малая обратимая задача, затем работа с сайтом, и только после этого крупные проходы по теории.


Дополнение: ChatGPT с GitHub-интеграцией может использоваться для точечных репозиторных правок, прежде всего для `work/discourse.md`, `/project` и `/protocols`. Codex остаётся основным исполнителем тяжёлых задач, проверок, больших diff и PR.
