# Carry-forward summary

The next workflow should not continue old “10 honest pass” source-transfer cycles as the main method. Those cycles produced useful material but also damaged composition.

The correct next phase is:

1. Codex workspace initialization.
2. Source precedence enforcement.
3. Skeleton v3 creation.
4. Dossiers for deep anchors and specification cluster.
5. Baseline restoration for SPDD and Gas Town.
6. Drafting in batches.
7. Anti-catalog and anti-degradation audits.
8. Human gates only at semantic decisions.

The main danger is repeating the catalogue failure. The main defense is to make each part carry one lifecycle function and to restrict cases to clear roles.
