# C5 concept-first atlas plan update — 17 passes + final verification

## Изменение

`C5_THEORY_TO_TECHNICAL_ATLAS_TARGET_GROUP_PLAN.md` перестроен под новое решение о concept-first / concept atlas. План больше не трактует C5 как простой мост «теория → техническое приложение». C5 задаёт две траектории чтения: theory-first через поперечный SDLC-синтез и concept-first через самостоятельные статьи атласа по конкретным концепциям.

## Очередь

План содержит 17 рабочих проходов плюс отдельную финальную проверку package-run:

- P01 — первичный фрагмент: две траектории чтения.
- P02 — модель самостоятельной статьи атласа.
- P03 — карта будущих статей атласа.
- P04 — пограничные случаи маршрутизации материала.
- P05 — проверка четырёх перекосов атласа.
- P06 — свободное раскрытие источников.
- P07 — системное выравнивание.
- P08 — адресный проход: самостоятельная ценность atlas article.
- P09 — адресный проход: controlled repetition.
- P10 — адресный проход: двусторонняя навигация theory-first / concept-first.
- P11–P13 — три общих редакторских прохода без специальной повестки.
- P14–P15 — два языковых прохода.
- P16–P17 — два стилевых прохода.
- Финальная проверка — отдельный final record runner-а, не рабочий проход.
