# Future target-specific reinforcement passes update

Дата: 2026-06-11.

## Решение

Для ещё не построенных фрагментов `00`, `A10`, `C1`–`C4` добавлены адресные проходы усиления основной функции перед общей редакторской тройкой.

Это не замена общих редакторских проходов. Новая последовательность для этих планов:

```text
системное выравнивание
→ 1–3 адресных прохода усиления основной функции
→ 3 общих редакторских прохода без специальной повестки
→ языковые проходы
→ стилевые проходы
```

`C5_THEORY_TO_TECHNICAL_ATLAS_TARGET_GROUP_PLAN.md` намеренно не изменён: пользователь отдельно указал, что C5 будет обсуждаться отдельно после решения о concept-first atlas.

- `00_SPINE_MAP_TARGET_GROUP_PLAN.md`: 2 прохода — терминологический контракт под нагрузкой; проверка, что spine-map не стал оглавлением.

- `C1_SPECIFICATION_TO_PWG_TARGET_GROUP_PLAN.md`: 2 прохода — переход спецификации к единице работы; граница specification/PWG.
