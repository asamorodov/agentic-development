# Отчёт: протокол очередей prompt-ов для написания глав и подглав

Дата: 2026-06-10.

Создан документ `work/protocols/THEORY_WRITING_PROMPT_QUEUE_PROTOCOL.md`.

Он фиксирует, как для writing-пакетов строить очередь prompt-ов: первичный черновик, 2–4 прохода добавления фактуры, системное выравнивание, repair-pass, отдельные языковые и стилевые проходы, работа с внутренними и внешними источниками, immediate provenance, `<figure>`-кандидатами и linked-target-edit для нескольких целевых документов.

Документ намеренно лаконичен: он задаёт направления и границы, а не длинные бюрократические чек-листы.

- `work/theory-writing/WORKING_DOCUMENTS_MAP.md` получил раздел о протоколах пакетного написания теории.
