# Source accumulation protocol cleanup

Дата: 2026-06-08  
Статус: точечная очистка общего протокола и prompt методологического досье.

## Что изменено

1. Убрана часть английского связующего языка из протокола и prompt.
2. В протокол добавлено правило: если источник найден, но не прочитан, по нему нельзя делать содержательные выводы. Его нужно пометить как `непрочитанный / требует проверки`.
3. Уточнено имя файлов прохода: для `passes/*` используется безопасное имя документа без директорий и расширения.
4. Из prompt методологического досье убрано дублирование общего протокола в разделе “что считать хорошим результатом”.
5. Раздел про иллюстрации оставлен без смыслового ограничения, как просил пользователь.

## Что не менялось

- Логика иллюстраций.
- Разделение: общий протокол + жанровый prompt + внешний оркестратор.
- Superseded status старого pass-протокола and старого Stage 0.19 prompt.
