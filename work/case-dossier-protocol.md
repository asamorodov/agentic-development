# Case dossier protocol

For each anchor or major framework case, create a dossier before writing final prose.

Required sections:

1. Role in AI-driven SDLC.
2. Primary sources and links.
3. What lifecycle step this case illuminates.
4. Internal workflow / mechanism.
5. Artifacts produced.
6. Human decision points.
7. Evidence / validation / review mechanisms.
8. Failure modes and boundaries.
9. What this case explains better than other cases.
10. What must not be compressed.
11. How it should appear in the rebuilt theory.
12. What should move to source map / Handbook / Fieldbook instead.

For SPDD and Gas Town, include a baseline detail inventory from old site seed sections.
