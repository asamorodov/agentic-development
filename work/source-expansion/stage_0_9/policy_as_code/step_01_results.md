# Search step 1 results — OPA / policy as code basics

## Источники

- Open Policy Agent — general-purpose policy engine; decouples policy decision-making from enforcement; works across microservices, Kubernetes, CI/CD, API gateways and more.
- Conftest — validates configuration files using policies.
- Kyverno — Kubernetes-native policies for validation, mutation, generation and image verification.

## Оценка

Policy-as-code is a concrete bridge between governance and execution. It should not replace human governance, but it can turn some constraints into checked artifacts.
