# Search step 1 results — LLM-generated ADRs and design decisions

## Selected sources

- `Can LLMs Generate Architectural Design Decisions?` — LLMs can generate relevant ADD/ADR decisions from context but not at human level.
- `DRAFT-ing Architectural Design Decisions using LLMs` — combines retrieval/fine-tuning/few-shot over ADR corpus to draft decisions more effectively.
- `Using LLMs in Generating Design Rationale for Software Architecture Decisions` — LLM-generated rationales can include helpful arguments but also uncertain or misleading ones.

## Evaluation

LLMs can assist decision documentation, but decision provenance remains human-gated. This is directly relevant to using ChatGPT/Codex for ADR proposals.

## Impact on theory

Proposed ADR workflow should be framed as draft/review, not automatic acceptance.
