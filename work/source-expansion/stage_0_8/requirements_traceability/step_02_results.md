# Search step 2 results — LLM-assisted traceability

## Selected sources

- `TraceLLM` — uses prompt engineering and demonstration selection to extract trace links across requirements, design elements, tests and regulations; supports semi-automated workflows with human validation.
- `Enhancing Requirement Traceability through Data Augmentation Using LLMs` — uses LLMs to generate requirement-to-code trace links and improves model performance.
- VERIFAI / Formalising Software Requirements using LLMs — connects natural language requirements to formal specifications and traceability through implementation and verification.

## Evaluation

LLMs can assist traceability recovery, but this reinforces the need for explicit trace artifacts. If the traceability layer is implicit, agents reconstruct it unreliably.

## Impact on theory

Add semi-automated traceability as a supporting mechanism in Part III/X/XII, not a deep case.
