# Search step 1 results — developer productivity frameworks

## Selected sources

- SPACE framework — productivity should not be measured by activity alone; includes satisfaction/well-being, performance, activity, communication/collaboration, efficiency/flow.
- DevEx framework — feedback loops, cognitive load and flow state.
- DORA 2025 — AI amplifies the existing system, not automatically improving organizational performance.

## Evaluation

This supports a key warning: more code activity is not equal to software value. AI-driven SDLC should account for human cognitive load and review capacity.

## Impact on theory

Part I/Conclusion should include productivity framing. Part X/XI should treat human review capacity as scarce resource.
