# Dossier-backed completeness rule update

Дата: 2026-06-12.

Изменение: в `ATLAS_ARTICLE_PACKAGE_BLUEPRINT.md` и все atlas article target-group plans добавлено правило полноты переноса из досье и первоисточников.

Смысл: досье является основной базой статьи, а не набором примеров для выборочного использования. Будущий article-writing package должен перенести в основной текст всю фактуру, которая укладывается в назначение статьи, либо явно дать disposition: transferred / partially transferred / not_relevant / deferred / belongs_to_neighboring_article / needs_external_check / rejected_with_reason.

Обновлены:

- P03: dossier coverage matrix;
- P04–P08: source-depth transfer-audit;
- P09–P10: свободный добор как добор недостающей фактуры и связности;
- Final: блокировка silent loss релевантного dossier/source material.
